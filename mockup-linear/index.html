PREGUNTAS

Lo que la gente siempre
nos pregunta.

¿Qué tan rápido se puede poner en marcha?
Setup 72 horas. Si confirmas hoy y nos envías acceso a tu Meta Business Suite, ARIA está respondiendo en WhatsApp el lunes. El onboarding lo lleva personalmente el fundador.
¿Puedo usar mi WhatsApp Business actual?
Sí. Conectamos tu número existente a la Meta Cloud API oficial. Tu historial, contactos y check azul se preservan. Cero riesgo de ban porque usamos la API verificada, no QR scraping.
¿ARIA puede equivocarse? ¿Qué control tengo?
ARIA opera con confirmación humana cuando detecta intent ambiguo. Tienes un drawer en el dashboard donde apruebas, rechazas o ajustas sus sugerencias antes de ejecutar. Para preguntas claras actúa autónoma. Para casos sensibles (cancelaciones de último minuto, reembolsos) te pregunta.
¿Qué pasa cuando se acaben los 20 spots Founding?
El plan público abre en Q3 2026 a $197 USD/mes. Los primeros 20 Founding Members conservan $97/mes locked-in de por vida con un precio Stripe separado. Sin compromiso hoy — confirmas tu spot por email cuando el producto esté al 100%.
¿Cuándo llega el plan Enterprise?
Q3 2026, solo por invitación. Para operaciones multi-sede, franquicias y grupos con 3+ sucursales o 500+ citas/mes. Stack custom construido contigo. Si te interesa, súmate a la lista de espera desde el footer.
¿Puedo cancelar cuando quiera?
Sí. Self-serve desde el dashboard, sección Billing. Un click, sin llamadas, sin retención forzada. Tu data se exporta automáticamente a CSV antes de cerrar la cuenta.
¿En qué países funciona?
República Dominicana, México, Estados Unidos y la mayoría de LATAM. WhatsApp Cloud API requiere que tu negocio esté verificado en Meta Business Suite — durante el onboarding te ayudamos con eso si no lo tienes.
¿Hay versión web del dashboard?
Hoy el dashboard es nativo para macOS y Windows (descarga directa). Versión web está en roadmap para Q4 2026. Si trabajas en equipo y necesitas acceso desde cualquier dispositivo, contáctanos para discutir Enterprise.
¿Qué integraciones tiene hoy y cuáles vienen?
Hoy: WhatsApp Cloud API oficial, Meta Business, email automatizado vía Resend, n8n workflows. Próximo trimestre: Google Calendar bidireccional, Stripe payments, Calendly imports, Slack notifications. API pública v1 para clientes Enterprise.
