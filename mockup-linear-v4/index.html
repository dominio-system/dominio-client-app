Sistema de color

Neutro como base.
Verde solo cuando significa algo.

El verde no es un acento decorativo: es señal semántica. Aparece cuando ARIA está funcionando, cuando una cita se confirmó, cuando una métrica crece. Eso refuerza el mensaje del producto sin saturar la marca.

Brand · Black
CTAs primarios, headlines, logo. El color de mayor peso visual.
#0a0a0a
Neutral · Off-white
Sections, cards, backgrounds elevados. Genera respiro.
#fafafa · #ffffff
Text · 3 niveles
Jerarquía sin colores: title / body / muted. Legibilidad WCAG AA.
#0a0a0a · #525252 · #a3a3a3
Success · Emerald
Semántica: "esto funciona". Solo states + indicadores.
#10b981

Reglas de uso del verde

Usar para
  • ● "Sistema activo" status dot (navbar, footer)
  • ✓ Checkmarks en pricing y features
  • ↑ Métricas que crecen (deltas positivos)
  • ● "ARIA online" indicator
  • Badge "TRIAL · DÍA N/14" en dashboard
  • Confirmaciones de acción ("cita agendada")
No usar para
  • Botones CTA primarios
  • Headlines o highlight de hero
  • Logo o tagline
  • Borders de cards generales
  • Backgrounds amplios de sección
  • Texto de body o párrafos
